## Hacking

### Dependencies

    - [Leiningen](http://leiningen.org/)
    - [Guard](http://guardgem.org/)

### Running server

    ```bash
    $ lein ring server
    ```

### compiling client code

    ```bash
    $ lein cljsbuild auto
    ```

### compiling templates

    ```bash
    $ guard
    ```
